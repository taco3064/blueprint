# Adoption Showcase

One theme — a tiny top-stories reader — adopted across every supported stack.
Each example uses the same pieces: a **page**, a **container** that assembles
it, a **component** leaf, a **context** provider, a **hook / composable** for
state, and a **service / lib** for data. Compare how blueprint shapes the same
app in each stack.

Every structure below is regenerated on each deploy by running the real CLI
against the committed example projects — `blueprint inspect` must pass for the
section to build, so what you see is always the current, true output.
